Multi-Document RAG Chatbot

Project Overview:
This project is a chatbot that can answer questions from multiple PDF documents using AI.

Features:
- Upload multiple PDFs
- Extract text from PDFs
- Store data using ChromaDB
- Ask questions and get answers

Technologies Used:
- Python
- Streamlit
- LangChain
- Groq API
- ChromaDB

How to Run:
1. Add PDFs into the pdfs folder
2. Run: python vectorize_documents.py
3. Run: streamlit run app.py
4. Open browser and ask questions
